# Contributing

Contributions, corrections, and new experiments are welcome.

## Getting set up

```bash
git clone https://github.com/USERNAME/ca-dynamics
cd ca-dynamics
pip install -e ".[dev,notebooks]"
pytest -q                 # should report all tests passing
python examples/verify.py # cross-checks output against the book
```

## Guidelines

- **Tests.** New functionality in `cadyn/` should come with a test in
  `tests/`. Numerical results that appear in the book should be verified
  in `examples/verify.py`.
- **Figures and notebooks.** Figures are generated by the scripts in
  `examples/`; notebooks are generated from the `build_*.py` scripts in
  `notebooks/` and then executed. Don't hand-edit the `.ipynb` files —
  edit the builder and re-run it, then re-execute:
  ```bash
  cd notebooks && python build_ch1_2.py   # (etc.)
  jupyter nbconvert --to notebook --execute --inplace 0*.ipynb
  ```
- **Style.** Plain NumPy/SciPy/Matplotlib/NetworkX; keep dependencies
  light. Match the surrounding code's docstring style.
- **Book text.** The LaTeX source lives in `latex/`. Corrections to the
  prose are welcome; please keep the `[2026 update]` boxes for
  forward-looking notes.

## Reporting problems

Open an issue describing what you expected and what happened. For a
numerical discrepancy, include the smallest snippet that reproduces it.
